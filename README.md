# Hotel Lobby Dashboard — Demo Template

A sleek, modern lobby display for hotels and guesthouses. Built with React, TypeScript, Vite, and Tailwind CSS v4.

## Features

- **Live Clock** — Large time/date overlay on hero image
- **Weather Widget** — Current conditions + 6-day forecast (mock data)
- **News Carousel** — Rotating headlines with images
- **Public Transport** — Bus/train departures with connections
- **Guest Welcome** — Personalized welcome messages
- **Fully Responsive** — Works on any screen size

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Tech Stack

- React 19 + TypeScript
- Vite 5
- Tailwind CSS v4 (Vite plugin)

## Customization

All config is in `src/config.ts`:
- Location name
- Guest names
- Rotation timings
- Image paths

Bus schedules are in `src/data/bus.ts`.
